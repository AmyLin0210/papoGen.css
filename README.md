# papoGen.css
CSS template for papoGen.

# Compile 

* global

```
# Install sass on your develop env.
[sudo] npm install -g sass

# Run the compile script.
./scss.sh
```

* local

```
# build
npm run build

# generate
npm run gen
```

# Test 

* Open the `docs/index.html` by your browser
    * Mostly recommend using `google-chrome`
