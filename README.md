# papoGen.css
CSS template for papoGen.

# Compile 

* global

```
# install sass on your develop env.
[sudo] npm install -g sass

# then run the compile script.
./scss.sh
```

* local

```
# build
npm run build

# generate
npm run gen
```

# Test 

* Open the `app.html` under `test/` by your browser
    * Mostly recommend using `google-chrome`
